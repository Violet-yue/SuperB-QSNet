# SuperB-QSNet
The code is of Beyond Spatial Priors: Spectral-Aware Hyperspectral Anomaly Detection Network via Heterogeneous–Homogeneous Super-Band and Quaternionic Sparse Representation.


Step 1: Unzip the compressed folder.

Step 2: In the extracted folder "SuperB-QSNet", run the Untitled.m program. You can import different dataset files (dataset1 to dataset6) and modify the relevant code accordingly to output preliminary sparse component files (S1 to S6). 
(Ensure the folder path is correctly set.)

Step 3: Open the standalone_detection sub-project folder in a Python environment and run:
python main.py --num 1 --train --epochs 0
(You can adjust the relevant parameter settings as needed.)


X. Wang, Y. Sun, T. Tang and L. Lei, "Beyond Spatial Priors: Spectral-Aware Hyperspectral Anomaly Detection Network via Heterogeneous–Homogeneous Super-Band and Quaternionic Sparse Representation," in IEEE Transactions on Geoscience and Remote Sensing, vol. 63, pp. 1-17, 2025, Art no. 5536217, doi: 10.1109/TGRS.2025.3639603.
